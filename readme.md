## About 💮

## Requirement 📷

- php version 7 or above
- MariaDB or Mysql
- server (apache)

## Run Local 💻

1. Clone repo ini menggunakan git
2. Letakan di folder server (htdoc)
3. Import database ke mysql
4. Done 🙂 🎊
